# Documentation

This is documentation about the development and design of `monodocs`.
For an overview of the tool and how to use it, see the [README](../README.md) in the repository root.

These files are AI-facing repository documentation and are maintained in English. Intentionally localized
user-facing content—such as the root [`README.md`](../README.md), the [`site/`](../site/) content, and the
[`examples/en/`](../examples/en/) and [`examples/ja/`](../examples/ja/) showcases—remains in its target language.
