# 脚注（GFM 拡張）

`[^id]` で脚注参照、`[^id]: ...` で本文を書きます。脚注の ID は単一 HTML 内でページごとに一意化され、
ページをまたいでも衝突しません。参照や戻りリンクをクリックすると、ページを切り替えずに該当箇所へスクロールします。

**ソース:**

```markdown
本文中に脚注参照を置けます[^1]。複数置くこともできます[^note]。
同じ脚注を再利用することもできます[^1]。長い注釈も書けます[^long]。

[^1]: これが 1 つ目の脚注です。
[^note]: 名前付きラベルの脚注。**強調**や `code`、[リンク](https://example.com)も含められます。
[^long]:
    複数段落の脚注。

    インデントを続けると 2 段落目になります。
```
