# 深い見出しと目次

ページ内目次（右側）に出す見出しの深さは `toc.maxLevel` で調整できます。
既定では **h2〜h3** を目次に出します（h1 はページタイトル相当のため常に除外）。
`toc.maxLevel: 4` にすると h4 まで、`toc.maxLevel: 2` にすると h2 だけが目次に並びます。

見出しは目次に出るかどうかに関わらず本文には常に表示されるため、目次の深さを浅くしても
情報が消えるわけではありません（目次が長くなりすぎるのを防ぐための設定です）。
