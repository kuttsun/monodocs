= ブロック
:sd-order: 102

各項目は *ソース* を先に示し、その下に *表示（HTML 変換結果）* を並べます。

== ソースブロック（構文ハイライト）

`[source,言語]` を付けた `----` ブロックは shiki でハイライトされます。

*ソース:*

[source,asciidoc]
------
[source,python]
----
def fib(n):
    a, b = 0, 1
    for _ in range(n):
        a, b = b, a + b
    return a
----
------

*表示:*

[source,python]
----
def fib(n):
    a, b = 0, 1
    for _ in range(n):
        a, b = b, a + b
    return a
----

`// <1>` のようなコールアウトで注釈を付けられます。

*ソース:*

[source,asciidoc,subs="-callouts"]
------
[source,js]
----
function greet(name) { // <1>
  return `Hello, ${name}!`; // <2>
}
----
<1> 関数を定義する。
<2> テンプレートリテラルで返す。
------

*表示:*

[source,js]
----
function greet(name) { // <1>
  return `Hello, ${name}!`; // <2>
}
----
<1> 関数を定義する。
<2> テンプレートリテラルで返す。

== リスティングブロック / リテラルブロック

`----` は等幅でそのまま、`....` はリテラル（空白も保持）。

*ソース:*

[source,asciidoc]
------
----
リスティングブロック（等幅・そのまま表示）
----

....
リテラルブロック
  空白も保持されます。
....
------

*表示:*

----
リスティングブロック（等幅・そのまま表示）
----

....
リテラルブロック
  空白も保持されます。
....

== 例ブロック

`====` で本文や他のブロックをまとめます。

*ソース:*

[source,asciidoc]
------
====
例ブロック。本文や他のブロックをまとめます。
====
------

*表示:*

====
例ブロック。本文や他のブロックをまとめます。
====

== サイドバーブロック

`****` は補足情報の囲みです。

*ソース:*

[source,asciidoc]
------
****
サイドバーブロック。補足情報の囲み。
****
------

*表示:*

****
サイドバーブロック。補足情報の囲み。
****

== 引用ブロック / 詩ブロック

`[quote]` は引用、`[verse]` は改行・字下げを保持する詩ブロックです。

*ソース:*

[source,asciidoc]
------
[quote, 山田太郎, 単一ファイル配布の心得]
____
ドキュメントは分割して書き、配布は 1 ファイルに。
____

[verse, 作者名]
____
改行や
  字下げが
保持される詩ブロック。
____
------

*表示:*

[quote, 山田太郎, 単一ファイル配布の心得]
____
ドキュメントは分割して書き、配布は 1 ファイルに。
____

[verse, 作者名]
____
改行や
  字下げが
保持される詩ブロック。
____

== Admonition（注意書き）

`NOTE:` などのラベルで注意書きになります。

*ソース:*

[source,asciidoc]
------
NOTE: 単純な NOTE。

TIP: 役立つヒント。

IMPORTANT: 重要な注意。

WARNING: 警告。

CAUTION: 注意。
------

*表示:*

NOTE: 単純な NOTE。

TIP: 役立つヒント。

IMPORTANT: 重要な注意。

WARNING: 警告。

CAUTION: 注意。

`[NOTE]` + `====` のブロック形式なら複数段落や他のブロックを含められます。

*ソース:*

[source,asciidoc]
------
[NOTE]
====
ブロック形式の NOTE。複数段落や他のブロックを含められます。

* リストも
* 入れられます
====
------

*表示:*

[NOTE]
====
ブロック形式の NOTE。複数段落や他のブロックを含められます。

* リストも
* 入れられます
====

== Mermaid（[source,mermaid]）

`[source,mermaid]` ブロックは図として描画されます。

*ソース:*

[source,asciidoc]
------
[source,mermaid]
----
graph TD
  A[AsciiDoc] --> B[Page モデル]
  B --> C[単一 HTML]
----
------

*表示:*

[source,mermaid]
----
graph TD
  A[AsciiDoc] --> B[Page モデル]
  B --> C[単一 HTML]
----

== パススルー（生 HTML）

`++++` で囲むと生 HTML をそのまま埋め込みます（信頼できる入力前提）。

*ソース:*

[source,asciidoc]
------
++++
<div style="padding:0.5rem;border:1px dashed #888;border-radius:6px;">
  パススルーで埋め込んだ生 HTML（信頼できる入力前提）。
</div>
++++
------

*表示:*

++++
<div style="padding:0.5rem;border:1px dashed #888;border-radius:6px;">
  パススルーで埋め込んだ生 HTML（信頼できる入力前提）。
</div>
++++
