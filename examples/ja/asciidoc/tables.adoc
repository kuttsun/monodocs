== 基本の表

`cols` で列幅と寄せ（`^` 中央 / `>` 右寄せ）、`options="header"` で 1 行目をヘッダーに、`.タイトル` でキャプションを付けます。

*ソース:*

[source,asciidoc]
------
.対応状況
[cols="1,^1,>1", options="header"]
|===
| 機能 | Markdown | AsciiDoc

| 見出し
| ✅
| ✅

| 表
| ✅
| ✅

| コードハイライト
| shiki
| shiki
|===
------

*表示:*

.対応状況

== セル内の書式と複数行

セル内で `*太字*` や `code` を使えます。`a|` プレフィックスでセルに AsciiDoc ブロック（リストなど）を入れられます。

*ソース:*

[source,asciidoc]
------
[cols="1,2", options="header"]
|===
| 項目 | 説明

| 強調
| セル内で *太字* や `code`、_イタリック_ が使えます。

| 箇条書き
a|
セルに AsciiDoc ブロック（リストなど）を入れるには `a` プレフィックスを使います:

* 項目 1
* 項目 2
|===
------

*表示:*

[cols="1,2", options="header"]
|===
| 項目 | 説明

|===

== CSV 形式

`format=csv` でカンマ区切りのデータから表を作れます。

*ソース:*

[source,asciidoc]
------
[%header, format=csv]
|===
名前,役割
core,変換処理の中核
cli,コマンドライン
|===
------

*表示:*

[%header, format=csv]
|===
名前,役割
core,変換処理の中核
cli,コマンドライン
|===
