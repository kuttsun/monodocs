# リスト

## 箇条書き（順序なし）

`-` / `*` / `+` のいずれも使えます。

- 項目 A
- 項目 B
  - ネスト B-1
  - ネスト B-2
    - さらにネスト
- 項目 C

## 番号付き（順序あり）

1. 最初
2. 次
3. その次
   1. ネスト 3-1
   2. ネスト 3-2

開始番号を指定:

5. 5 から
6. 6
7. 7

## タスクリスト（GFM 拡張）

- [x] 完了したタスク
- [ ] 未完了のタスク
  - [x] 子タスク（完了）
  - [ ] 子タスク（未完了）

## ゆるいリストと詰まったリスト

詰まった（tight）リスト:

- 行間が詰まる
- 2 行目

ゆるい（loose）リスト（項目間に空行）:

- 段落として描画される

- 項目の間に空行があると緩くなる

## 定義的な内容

順序なしリストの各項目に段落や引用を含められます。

- 項目に複数段落を含む例。

  これは同じ項目の 2 段落目です（インデント）。

  > 項目内の引用。
