# 脚注（GFM 拡張）

本文中に脚注参照を置けます[^1]。複数置くこともできます[^note]。
同じ脚注を再利用することもできます[^1]。

脚注の番号と本文は単一 HTML 内でページごとに一意な ID になり、ページをまたいでも衝突しません。
脚注参照や戻りリンクをクリックすると、ページを切り替えずに該当箇所へスクロールします。

長い注釈も書けます[^long]。

[^1]: これが 1 つ目の脚注です。
[^note]: 名前付きラベルの脚注。**強調**や `code`、[リンク](https://example.com)も含められます。
[^long]:
    複数段落の脚注。

    インデントを続けると 2 段落目になります。
