# インストール

## 必要要件

- Node.js 22 以上
- pnpm

## 手順

| ステップ | コマンド                                      |
| -------- | --------------------------------------------- |
| 取得     | `pnpm add -D single-docs`                     |
| ビルド   | `single-docs build ./docs -o ./dist/out.html` |

- [x] Node.js を用意する
- [ ] ドキュメントを書く
