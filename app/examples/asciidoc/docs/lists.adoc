= リスト
:sd-order: 2

== 箇条書き（順序なし）

* 項目 A
* 項目 B
** ネスト B-1
** ネスト B-2
*** さらにネスト
* 項目 C

== 番号付き（順序あり）

. 最初
. 次
. その次
.. ネスト 2-1
.. ネスト 2-2

== チェックリスト

* [x] 完了したタスク
* [ ] 未完了のタスク
* [*] 完了（別記法）

== 説明リスト（description list）

CPU:: 中央処理装置。
メモリ:: 主記憶。
ストレージ::
  補助記憶。複数行の説明も書けます。

== 用語と入れ子

オペレーティングシステム::
  Linux:::
    オープンソースの OS。
  macOS:::
    Apple の OS。

== リスト項目に複数ブロック

* 項目に複数のブロックを含む例。
+
これは継続行（`+`）でつないだ同じ項目の段落です。
+
[source,bash]
----
echo "リスト項目内のコードブロック"
----
