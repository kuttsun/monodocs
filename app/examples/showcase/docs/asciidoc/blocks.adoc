== ソースブロック（構文ハイライト）

[source,python]
----
def fib(n):
    a, b = 0, 1
    for _ in range(n):
        a, b = b, a + b
    return a
----

コールアウト付き:

[source,js]
----
function greet(name) { // <1>
  return `Hello, ${name}!`; // <2>
}
----
<1> 関数を定義する。
<2> テンプレートリテラルで返す。
