=== レベル 2

==== レベル 3

===== レベル 4

== 段落と改行

これは段落です。空行で段落を区切ります。

行末に `+` を置くと +
強制改行になります。

[.lead]
リード段落（`[.lead]` ロール）。

== テキスト書式

* 太字: *bold* と **un**constrained
* イタリック: _italic_ と __un__constrained
* 等幅: `monospace`
* 太字イタリック等幅: `*_重ね掛け_*`
* ハイライト: #highlighted#
* 上付き/下付き: E = mc^2^ / H~2~O
* 引用符: "`曲がった二重引用符`" と '`単一引用符`'

== 置換（テキスト置換）

(C) (R) (TM) は記号に、矢印 -> => <- <= はそのまま、`...` は省略記号になります。
