= リスト
:sd-order: 2

各項目は *ソース* を先に示し、その下に *表示（HTML 変換結果）* を並べます。

== 箇条書き（順序なし）

`*` の数でネストします。

*ソース:*

[source,asciidoc]
------
* 項目 A
* 項目 B
** ネスト B-1
** ネスト B-2
*** さらにネスト
* 項目 C
------

*表示:*

* 項目 A
* 項目 B
** ネスト B-1
** ネスト B-2
*** さらにネスト
* 項目 C

== 番号付き（順序あり）

`.` で番号付き、`..` でネストします。

*ソース:*

[source,asciidoc]
------
. 最初
. 次
. その次
.. ネスト 2-1
.. ネスト 2-2
------

*表示:*

. 最初
. 次
. その次
.. ネスト 2-1
.. ネスト 2-2

== チェックリスト

`[x]` / `[ ]` でチェックボックスになります。

*ソース:*

[source,asciidoc]
------
* [x] 完了したタスク
* [ ] 未完了のタスク
* [*] 完了（別記法）
------

*表示:*

* [x] 完了したタスク
* [ ] 未完了のタスク
* [*] 完了（別記法）

== 説明リスト（description list）

`用語:: 説明` で用語と説明の組を作ります。

*ソース:*

[source,asciidoc]
------
CPU:: 中央処理装置。
メモリ:: 主記憶。
ストレージ::
  補助記憶。複数行の説明も書けます。
------

*表示:*

CPU:: 中央処理装置。
メモリ:: 主記憶。
ストレージ::
  補助記憶。複数行の説明も書けます。

== 用語と入れ子

`::` / `:::` でレベルを変えて入れ子にできます。

*ソース:*

[source,asciidoc]
------
オペレーティングシステム::
  Linux:::
    オープンソースの OS。
  macOS:::
    Apple の OS。
------

*表示:*

オペレーティングシステム::
  Linux:::
    オープンソースの OS。
  macOS:::
    Apple の OS。

== リスト項目に複数ブロック

継続行（`+`）で同じ項目に段落やブロックをつなげられます。

*ソース:*

[source,asciidoc]
------
* 項目に複数のブロックを含む例。
+
これは継続行（`+`）でつないだ同じ項目の段落です。
+
[source,bash]
----
echo "リスト項目内のコードブロック"
----
------

*表示:*

* 項目に複数のブロックを含む例。
+
これは継続行（`+`）でつないだ同じ項目の段落です。
+
[source,bash]
----
echo "リスト項目内のコードブロック"
----
