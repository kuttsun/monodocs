# 応用：3 階層目のサンプル

このフォルダ（`markdown/advanced/`）は **3 階層目** に置かれたページ群です。
入力を深くフォルダ分割しても、サイドバーはフォルダ構造をそのままツリーにします。

- 階層が深くなってもサイドバーで畳む／開くで辿れます（`sidebar.collapseDepth` で
  「この階層より深いものは既定で畳む」設定にできます。**隠す**のではなく**畳む**だけなので、
  どのページにも常に到達できます）。
- 同じ階層のページへもリンクできます: [深い見出しのページ](deep-headings.md)

## 深いフォルダでもリンクは追従する

3 階層目から 1 階層目へ戻るリンクも、単一 HTML 内の hash route に変換されます。

- 1 つ上のフォルダ: [Markdown サンプル](../index.md)
- ルート: [ショーケース](../../index.md)
